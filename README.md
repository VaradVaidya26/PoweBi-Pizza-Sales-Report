# Pizza Sales Analysis Dashboard (Power BI)

 ## Overview

This project presents an interactive Pizza Sales Dashboard built using Microsoft Power BI, designed to analyze sales performance, customer trends, and product insights.

The dashboard provides actionable insights into revenue, order patterns, and best-selling pizzas to support data-driven decision-making.

## Key Features
- Revenue analysis (daily, monthly trends)
- Top & bottom performing pizzas
- Order quantity and sales distribution
- Peak order times analysis
- Category & size-based performance breakdown

- KPI metrics: Total Revenue, Orders, Average Order Value

## Tools & Technologies
- Microsoft Power BI
- Data Modeling
- DAX (Data Analysis Expressions)
- Data Cleaning & Transformation

## Dataset Source: Pizza sales dataset (CSV)
Includes:
- Order details
- Pizza categories & sizes
- Pricing & quantities

## Key Insights
- Highest revenue generated during weekends and evenings
- Classic and Supreme categories drive the most sales
- Certain pizzas consistently underperform → potential optimization
- Peak ordering hours help in staffing & inventory planning

## How to Use
- Download the .pbix file from this repository
- Open in Microsoft Power BI Desktop
- Interact with filters, slicers, and visuals

## Project Objective
To demonstrate:
- End-to-end data analysis workflow
- Business insight generation
- Interactive dashboard design
